# TypeScript マスターへの道：基礎から応用まで

TypeScript は、JavaScript に静的型付けを追加した強力なプログラミング言語です。

## TypeScript の基本

### 1. 型注釈

TypeScript の最も基本的な機能は型注釈です：

```typescript
let message: string = "Hello, TypeScript!";
let count: number = 42;
let isActive: boolean = true;
```
